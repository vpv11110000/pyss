40 Например, блок
41 
42 assemble.Assemble(sgm, countTransact=2,label="Assemble001")
43 
44 объединяет каждые два транзакта одного семейства.
45 
46 Блок Assemble всегда принимает транзакты.
47 Первый вошедший в блок член семейства остается ждать в нем прибытия
48 других членов этого же семейства. Каждый следующий член семейства,
49 входящий в блок, уничтожается и количество членов семейства,
50 подлежащих объединению, уменьшается на единицу.
51 
52 Если количество членов семейства, подлежащих объединению,
53 станет равным нулю, то первый вошедший в блок член
54 семейства пытается войти в следующий блок.
55 
56 Если транзакт не принадлежит ожидаемому семейству,
57 то он не обрабатывается, проходит транзитом.
58 
59 Если countTransact равен 0 или 1, то транзакт проходит транзитом.
60 
61 Атрибуты блока Assemble (в дополнение к атрибутам block.Block):
62 bl = Assemble(...)
63 bl[COUNT_TRANSACT] - количество объединяемых транзактов
64 bl[ITEMS] - хранилище данных сбора транзактов (много семейст - много сборок),
