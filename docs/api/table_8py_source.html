16  """Объект таблицы
17 
18 Таблица предназначена для сбора данных от элементов модели.
19 
20 Аргумент таблицы (аналог оси 'x') разбивается на countInt интервалов.
21 Ширина интервала задаётся параметром widthInt.
22 Левая граница интервала принадлежит интервалу.
23 Правая граница интервала не принадлежит интервалу.
24 Правая граница первого интервала задаётся параметром limitUpFirst.
25 Аргумент таблицы вычисляется вызовом функции, хранимой в атрибуте ARG_FUNC.
26 Значение аргумента таблицы вычисляется в блоке tabulate.Tabulate вызовом
27 функции, хранимой в атрибуте VAL_FUNC блока Tabulate.
28 
29 Таблица добавляется в модель с помощью вызова метода addTable контейнера pyss_model.PyssModel.
30 
31 Args:
32  ownerModel - объект модели-владельца
33  tableName - имя таблицы
34  argFunc=None - функция определения аргумента таблицы. Сигнатура argFunc(owner,transact).
35  limitUpFirst=None - правая граница первого интервала аргумента таблицы
36  widthInt=None - ширина интервалов
37  countInt=None - количество интервалов
38 
39 
40 Например, объект
41 
42 table.Table(tableName="T_1", argFunc=argFunc_T_1, limitUpFirst=1.0, widthInt=1.0, countInt=50)
43 
44 определяет таблицу T_1, аргумент таблицы определяется функцией argFunc_T_1, первый интервал ограничен сверху значением 1.0, ширина интервала 1.0, количество инетрвалов 50.
45 
46 См. также tabulate.py.
47 
48 Атрибуты объекта Table (в дополнение к атрибутам pyssobject.PyssObject):
49 tbl = Table(...)
50 tbl[TABLENAME] - имя таблицы.
51 tbl[TITLE] - дубль: имя таблицы.
52 tbl[DISPLAYING] - true - отображать результат обработки данных таблицы
53 tbl[ARG_FUNC] - функция вычисления аргумента таблицы, хранит значение параметра argFunc
54 tbl[LIMITUPFIRST] - правая граница первого интервала аргумента таблицы
55 tbl[WIDTHINT] - ширина интервалов
56 tbl[COUNTINT] - количество интервалов
57 tbl[TOTAL] - сумма всех значений вычисленных функции, хранимой в атрибуте VAL_FUNC блока Tabulate. Очень полезно, если значения аргемента таблица всехга равны, например, единице.
58 tbl[INTERVALS] - множество интервалов
59 tbl[MIN_BOUND] - левая граница таблицы
60 tbl[MAX_BOUND] - правая граница таблицы
61 tbl[LIST] - список правых границ всех интервалов
62 tbl[ACTION_FUNC] - функция занесения данных в таблицу, в зависимости от параметров может различаться для разных таблиц
63 tbl[ACTION_VIEW] - функция формата вывода, в зависимости от параметров может различаться для разных таблиц
64 tbl[LIST] - список правых границ всех интервалов
65 tbl[INTERVALS][POSITIVE_INFINITY] - величина значения справа от правой границы таблицы
66 tbl[INTERVALS][NEGATIVE_INFINITY] - величина значения слева от левой границы таблицы
67
